## Installation

OS X, Linux & Windows:

Cloning this repo
```sh
git clone https://github.com/JonathanRangelB/platzi-video
```

## Development setup

You can run the application with the following commands:

Installing dependencies
```sh
npm install
```

Run the project
```sh
npm start
```
